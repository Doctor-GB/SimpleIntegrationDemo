Описание
Консольное приложение на языке C#, демонстрирующее базовые принципы интеграции программных модулей. Приложение рассчитывает итоговую стоимость заказа на основе списка цен товаров и выводит результат в консоль.

Назначение модулей
Модуль расчёта стоимости заказа (OrderCalculator)
Отвечает за вычисление итоговой суммы. Принимает список цен, суммирует их и применяет скидку 10 процентов, если общая сумма превышает 1000 рублей. Возвращает финальную стоимость.
Модуль вывода результата (ResultPrinter)
Отвечает за отображение данных пользователю. Получает итоговую сумму и выводит её в консоль в отформатированном виде.
Интерфейс ICalculator
Обеспечивает абстракцию модуля расчёта. Позволяет использовать слабую связанность и при необходимости заменять реализацию калькулятора без изменения основного кода.

Способ взаимодействия модулей
Взаимодействие организовано в главном файле программы Program.cs. Сначала создаются экземпляры модулей. Затем формируется список цен товаров. Список передаётся в метод расчёта через интерфейс ICalculator. Полученная сумма передаётся в метод вывода результата, который отображает данные в консоли. Модули не зависят друг от друга напрямую, обмен данными происходит только через параметры методов.
